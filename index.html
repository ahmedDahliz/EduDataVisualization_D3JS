Data visualization using D3.js

This is a data visualization of an academic performance dataset (CSV format), that contain a bunch of information about student's absence, nationalities, educational stages, etc.
In total, this dataset contains 480 records with 17 attributes.
Github repository.

This visualization is as follows :

  • Pie chart: Number of parent school satisfaction (good & bad).
  • Donut chart: Number of times the student visits a course content by educational stage.
  • Stacked bar chart: Number of absences (under 7 days and above 7 days) by educational stage.
  • Horizontal lollipop chart: Number of students by nationality.
  • Bar chart: Number of raised hands per topic.

About the dataset

Students Academic Performance Dataset is an educational data set which is collected from learning management system (LMS) called Kalboard 360. Kalboard 360 is a multi-agent LMS, which has been designed to facilitate learning through the use of leading-edge technology.
Such system provides users with a synchronous access to educational resources from any device with Internet connection.
